# Getting Started

## Installation

you do need to have a couple of other things installed first.
* Node.js
* Git
### Install Hexo

```
$ npm install -g hexo-cli
```

# Git

## clone
```
$ git clone git@github.com:sam40305sam/home.git
```
And u will get my Files.~~~

***Don't Forget the command as shown below.***
```
cd home
npm install
```
# Build Project

## build hexo
```
$ hexo g
```

## server

```
$ hexo server
```
